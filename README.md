## 4. 기능
- 튜터 관리: 사용자는 튜터를 생성, 수정, 삭제할 수 있습니다.
- 메시지 교환: 채팅방에서 실시간으로 메시지를 주고받을 수 있습니다.
- 파일 관리: 사용자는 어시스턴트에 파일을 첨부하거나 제거할 수 있습니다.
- 채팅 스레드 생성 및 관리: 새로운 채팅 스레드를 시작하고 이전 스레드를 조회할 수 있습니다.
